IMPORTANT INFO
## starting the project
First open the terminal on vs studio or the ide that you are using
Second cd to the frontend folder .\study-groups-frontend\
Third install the npm dependencies with the "npm install" command.
Fourth run the program with the command npm start , this will run the program on http://localhost:3000/ .
